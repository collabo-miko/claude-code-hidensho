# Claude Code 社内指南書

シニアエンジニア向け Claude Code 実践ガイド & ハンズオンサンプル集。

基本操作からセキュリティ・コスト管理まで、Claude Code を実務で使いこなすために必要な知識を全30セクションで体系的にまとめています。

## 構成

```
claude-code-guide.md    # 指南書本体（全30セクション / 約2,600行）
samples/                # ハンズオンサンプル集（8テーマ）
```

### 指南書の章立て

| 編 | セクション | 内容 |
|----|-----------|------|
| 基本編 | 1-6 | Claude Code とは、セットアップ、基本操作、コマンド、ショートカット、CLI フラグ |
| 運用編 | 7-11 | コアプラクティス、プロンプトテクニック、権限モード、モデル選択、コンテキスト管理 |
| 応用編 | 12-20 | CLAUDE.md、MCP、Hooks、スキル、IDE連携、設定体系、サブエージェント |
| 実践編 | 21-24 | ワークフロー、アンチパターン、CI/CD連携、コスト管理 |
| ガバナンス編 | 25-30 | 社内ルール、セキュリティ、脅威モデル、インシデント事例、トラブルシューティング |

## 使い方

### 指南書を読む

`claude-code-guide.md` を順に読み進めてください。各セクションは独立しているので、必要な箇所から読むこともできます。

### ハンズオンを試す

```bash
# サンプルディレクトリに移動して Claude Code を起動
cd samples/04_hooks
claude

# 「README.md の演習を順番に進めたい」と伝えると Claude がガイドしてくれます
```

各サンプルは独立しており、好きな順番で試せます。

## 前提条件

- Claude Code がインストール済みであること（`npm install -g @anthropic-ai/claude-code`）
- 認証が完了していること（`claude` を初回起動してログイン）

## 注意事項

- `.env` ファイルはリポジトリに含まれていません（`.env.example` を参考に作成してください）
- `07_security/` の脆弱なコードは学習目的のサンプルです
- `06_mcp/` は外部サービスへの接続が必要な場合があります
