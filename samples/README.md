# Claude Code 指南書 ハンズオンサンプル集

各ディレクトリで `cd` して `claude` を起動すると、そのディレクトリの `.claude/` 設定が適用されます。
各章の内容を実際に手を動かしながら学べます。

## サンプル一覧

| ディレクトリ | 対応セクション | 学べること |
|-------------|---------------|-----------|
| `01_basic_setup/` | セクション3: 基本操作 | セッション管理、非対話モード、コンテキスト操作 |
| `02_permissions/` | セクション9: 権限モード | deny/allow ルール、権限モード切替、ツール制御 |
| `03_claude_md/` | セクション12: CLAUDE.md | ルール記述、パス限定、ディレクトリ別ルール、ファイル参照 |
| `04_hooks/` | セクション14: Hooks | PreToolUse/PostToolUse/Notification フック、exit code |
| `05_skills/` | セクション15: スキル | スキル作成、動的コンテキスト注入、オンデマンドロード |
| `06_mcp/` | セクション13: MCP | サーバー追加・管理、.mcp.json、コンテキスト影響 |
| `07_security/` | セクション26-27: セキュリティ | セキュリティ設定、脆弱性修正、Unicode チェック |
| `08_cost_optimization/` | セクション24: コスト管理 | トークン節約、モデル切替、エフォート調整、自動コンパクション |

## 使い方

```bash
# 1. サンプルディレクトリに移動
cd samples/04_hooks

# 2. Claude Code を起動（ディレクトリの .claude/ 設定が自動適用）
claude

# 3. README.md の演習に従って操作
# 「README.md の演習を順番に進めたい」と伝えると Claude がガイドしてくれます
```

## 注意事項

- 各サンプルは独立しています。好きな順番で試せます
- `.env` ファイルはダミーデータです（実際のシークレットは含まれていません）
- セキュリティサンプル（07）の脆弱なコードは学習目的です
- MCP サンプル（06）は外部サービスへの接続が必要な場合があります
