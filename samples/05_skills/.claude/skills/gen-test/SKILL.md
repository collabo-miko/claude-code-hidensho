# テスト生成手順

1. `src/` ディレクトリ内のソースファイルを読み取る
2. 各エクスポート関数に対してテストケースを作成する:
   - 正常系: 期待される入力と出力
   - 境界値: 空文字列、0、null、undefined
   - 異常系: 不正な型、範囲外の値
3. テストファイルを `tests/` ディレクトリに作成する
4. `npx jest` でテストを実行して結果を確認する

IMPORTANT: テストを通すためにプロダクションコードを変更しないこと
